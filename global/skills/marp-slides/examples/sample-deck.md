# Assessment Scope

- Reviewed 14 core business processes across operations, finance, and client delivery
- Interviewed 8 team members from leadership to frontline staff
- Audited data quality across 4 primary systems (CRM, accounting, project management, comms)
- Benchmarked against Optimi's Four Legs of AI Readiness framework

---

<!-- _class: two-column -->

# The Four Legs Framework

- **Clean Data** — Is your information accurate, accessible, and structured?
- **Documented Processes** — Are workflows written down, or locked in people's heads?
- **Decision Governance** — Who decides what, and how are decisions tracked?

<!-- column break -->

- **Team Capacity** — Does your team have bandwidth and willingness to adopt new tools?
- Each leg must be solid before adding AI weight
- Think of it as building foundations before adding a second storey

<!-- Speaker notes: This is the core of Optimi's methodology. We diagnose before we prescribe. No point recommending AI tools if the data feeding them is unreliable. -->

# Recommendations
## Your path to AI readiness

---

<!-- _class: two-column -->

# Phase 1: Foundations (Months 1-3)

### Data Cleanup
- Deduplicate CRM records
- Establish data entry standards
- Create a single project status dashboard

### Process Documentation
- Document top 6 critical workflows
- Assign process owners
- Build lightweight SOPs

<!-- column break -->

### Quick Wins
- Automate 3 manual data-entry tasks
- Set up CRM validation rules
- Create shared templates for recurring work

### Team Development
- AI literacy workshop for full team
- Identify 2-3 internal champions
- Set up feedback loops for new tools

<!-- Speaker notes: Phase 1 is all about building the foundations. No AI tools yet — just getting the basics right so that when we do introduce AI, it has clean data and clear processes to work with. -->

---

# Phase 2: Smart Automation (Months 4-6)

- Deploy AI-assisted document drafting for client communications
- Automate client intake and onboarding workflow
- Set up intelligent email routing and prioritisation
- Implement AI-powered data quality monitoring
- Begin predictive workload planning pilot

> Each automation saves an estimated 4-8 hours per week once operational

<!-- Speaker notes: This is where the investment starts paying off. These aren't speculative AI features — they're practical automations built on the clean foundations from Phase 1. -->
